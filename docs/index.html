🏗Pipeline Overview

Raw CRM Data → Feature Engineering (40+) → StandardScaler ↓ XGBoost + LightGBM → Stacking Ensemble (RF meta) ↓ SHAP Explainability + Drift Detection (KS/PSI) → FastAPI Scoring API

✨Key Features

Stacking Ensemble

XGBoost + LightGBM base models with Random Forest meta-learner via StackingClassifier (5-fold CV).

Feature Engineering

40+ behavioral features: login rates, support interaction ratios, engagement scores, rolling windows.

SHAP Explainability

Per-prediction SHAP values explain which features drove the churn score — essential for business trust.

Drift Detection

KS test + Population Stability Index (PSI) monitor feature distributions and trigger auto-retraining.

Real-time Scoring API

FastAPI endpoint scores any customer in <100ms using the serialized ensemble + scaler.

Docker + CI/CD

Containerized with Docker, deployed via CI/CD pipeline with model versioning via MLflow.

💻Stacking Ensemble Code

from sklearn.ensemble import StackingClassifier, RandomForestClassifier
  from xgboost import XGBClassifier
  from lightgbm import LGBMClassifier

  ensemble = StackingClassifier(
      estimators=[
          ("xgb",  XGBClassifier(n_estimators=300, max_depth=5,
                                 learning_rate=0.05, subsample=0.8)),
          ("lgbm", LGBMClassifier(n_estimators=300, max_depth=5,
                                  learning_rate=0.05, subsample=0.8)),
      ],
      final_estimator=RandomForestClassifier(n_estimators=50),
      cv=5,
  )
  ensemble.fit(X_train, y_train)
  print(f"F1:  {f1_score(y_test, ensemble.predict(X_test)):.4f}")
  print(f"AUC: {roc_auc_score(y_test, ensemble.predict_proba(X_test)[:,1]):.4f}")
